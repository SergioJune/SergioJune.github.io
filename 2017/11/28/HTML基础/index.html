现在我基本把HTML+css+JavaScript学完了，就等着我的笔记了。

所以我决定今周一天写一篇笔记！！！

好了，废话不多说了，写下今天所要写的笔记吧。

  • HTML是一种描述性的标记语言，并不是一种编程语言，想要装逼的小伙伴要记住了！！！

  • 基本语法为<标记符>中间这里填内容</标识符>—>前面的标识符为开始标志，后面的标识符为结束标志

    如 <p>基本语法</p>
  • 记住网页源代码的效果并不是网页的最终效果

  • html的文档结构如下：

    1. html标签：是个双标记，就是有开始和结束标志。它标识网页的开始与结束，所有的HTML元素都要放在这个标记里面
    2. head标签：双标记，标示网页文件的头部信息，如标题和搜索引擎等等
    3. title标签：双标记，标识网页的标题，放在head标签里面
    4. body标签：双标记，网页的主体部分，网页里显示的东西都放在里面

    简单网页就是这样组成的

      <!DOCTYPE html><!--这个用于声明文档类型，写在网页的第一行-->
<html>
    <head>
        <title>html网页</title>
        <meta charset="utf-8">
    </head>
    <body>

    </body>

</html>

  • meta标记：用于定义文件信息，便于搜索引擎查找，放在标记内。可以设置关键字（keywords），描述（description）和作者（author）等，格式如下

    1.<meta name="keywords" content="关键字"/><!--这个是设置关键字-->
2.<meta name="description" content="描述内容"/><!--这个是设置描述内容-->
3.<meta name="author" content="SergioJune"/><!--这个是设置作者-->
4.<meta http-equiv="content-type" content="text/html,charset=utf-8"/><!--这个是设置网站的编码方式-->
5.<meta http-equiv="1refresh" content="5,URL=跳转地址"/><!--这个是设置网页自动跳转，由于网页问题，我防止他自动跳转，我在前面的值加了个1，要的时候记得去掉  content的第一个值为开始自动跳转的等待时间，单位为秒-->

  • html的注释：

    <!--注释内容-->

  • pre标记：双标记，保留文本的原来格式，即文本格式原来是怎样，在网页输出也是怎样

  • 在网页中输出特殊字符：（只是常用的几个，其他的请自行查文档）

      &nbsp;<!--表示空格-->
&reg;<!--表示商标-->
&lt;<!--表示左边尖括号-->
&rt;<!--表示右边尖括号-->

  • HTML的列表：分为三种列表，分别是有序列表，无序列表和自定义列表，这三种均放在body标签内
    1.无序列表：由 ul和li标记组成 ，下面为格式和图片

    <ul type="circle"><!--无序列表-->
          <li><!--每一个小项-->
           把C语言给学完，
          </li>
          <li >
           有空也需要把Java复习一下，要不全忘记了
          </li>
          <li>
           每天学习两个英语音标（英语渣渣）
          </li>
          <li type="square"><!--这个单独设置了样式，所以显示图标为正方形-->
           有空的话可以把python学完
           </li>
   </ul>

    2.有序列表：由 o l和li标签组成 ，下面为格式和图片

        <ol type="A" start="26"><!--有序列表,里面有样式设置，type属性是指按什么排序，start属性是从那个开始-->
             <li><!~--这个是每个小项-->
              把C语言给学完，
             </li>
             <li >
              有空也需要把Java复习一下，要不全忘记了
             </li>
             <li>
              每天学习两个英语音标（英语渣渣）
             </li>
             <li >
              有空的话可以把python学完
              </li>
</ol>

    3.自定义列表：有 dl，dt和dd组成 ，而且这三个标签需要组合起来使用才有效

        <dl><!--自定义列表-->
          <dt><!--列表的主项-->
           数码
          </dt>
            <dd><!--该列表的仔细分类-->
               单反相机
               </dd>
            <dd>
               CPU
            </dd>
            <dd>
               笔记本 
            </dd>
          <dt>
           手机
          </dt>
            <dd>
               小米
            </dd>
            <dd>
               苹果
            </dd>
            <dd>
               魅族 
            </dd>
</dl>

  • img标签：单标记，属性有src（必写），alt，vspace，align等属性，下面展示格式加效果图

       <img src="img/1.jpg" align="left" hspace="100" height="180" width="180">
<!--src是图片的url地址值，alt是文字，当图片无法加载时显示的文字-->
<!--hspace是距离图片的水平像素，width和height是图片的宽和高-->

  • a标签：超链接标签，双标记，
    1.href属性;需要跳转的链接，可以是内部的，也可以是外部的，当该值是 # 时，表示的是空链接，就是点击无效果，用来调试的
    2.target属性：表示链接窗口的属性，有_self（在本窗口跳转），_blank（在新窗口跳转），_top，_parent属性
    3.title属性：链接提示性文字，当鼠标悬停时可以看到的文字
    4.name属性：给链接命名，定义锚时有用
    5.锚的用法和定义;使用命名锚记可以在文档中设置标记，这些标记通常放在文档的特定主题处或顶部。然后可以创建到这些命名锚记的链接，这些链接可快速将访问者带到指定位置。
    创建到命名锚记的链接的过程分为两步。首先，创建命名锚记，然后创建到该命名锚记的链接。 下面给出代码展示

       <a href="#py2">python进阶 </a><!--这个就是锚导航，点击后在本网页跳转，记得格式是 #锚名 -->
<a href="http://coding.imooc.com/class/136.html" target="_blank" title="点击跳转到相关课程" name="py3"></a><!--这个是命名锚记，就是上面点击后跳转的地方-->

    6.它还可以发送邮件，写什么反馈邮件时可以用到，格式为：

    <a href="mailto:邮件地址"></a>

    7.也可以下载文件，格式为：

    <a href="文件名"></a>
