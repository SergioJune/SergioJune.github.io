2. 寻找生成位置

上面知道了是哪个 cookie 反爬的，最简单的寻找就是全局搜索这个参数来看看有没有线索。

果然有，而且文件名字和请求的路径名有几分相似，按经验来说八九不离十就是这个了，点击进去看看。

有了上次的经验，我就直接在浏览器上格式化直接调试了，不把代码复制到本地运行了。
