安装docker及其简单操作

留言

Jan 20, 2019docker点击

1.下载地址

https://download.docker.com/win/stable/DockerToolbox.exe

点击上面链接直接就可以下载了，下载之后默认安装就可以了。

2.打开Docker Quickstart Terminal

就是打开这个软件。

注意：如果你之前安装了git的话，里面的目标需要自己手动配置下，需要手动找到给git的安装地址给更改

自己手动改下前面git的安装地址即可。

还有一个坑：就是点击之后，会下载一个东西：boot2docker.iso，这个需要科学上网。或者你可以直接在你安装的docker的地址复制这个文件到它指定的目录文件也可以。

把下面的文件复制到上上图中的指定地址也可以的。

复制之后再打开就可以了。

3.使用docker

1）查看版本
docker version

2)运行 hello world
docker run hello-world


这个会先在本地上查找有没有 hello-world 这个镜像，没有的话，就会在 docker hub 上下载到本地，然后在运行这个镜像。

我们还可以根据上面的提示运行一个 ubuntu 系统。
docker run -it ubuntu bash


这样子我们就进入这个 Ubuntu 系统内部了

退出这个系统输入exit即可

3)查看当前docker所运行的容器
docker ps -a

可以看到上面的容器的id、镜像、命令、状态和端口和容器名称（这个如果不指定名称的话会随机生成）

4)查看当前已下载的镜像
docker images

5)删除镜像
docker rmi ubuntu


Error response from daemon: conflict: unable to remove repository reference "ubuntu" (must force) - container 8b7a4c6c7a10 is using its referenced image 1d9c17228a9e
这里看到会报错，因为还有一个地方用到这个镜像，需要先删除8b7a4c6c7a10

删除8b7a4c6c7a10
docker rm 8b7a4c6c7a10

再删除镜像
docker rmi ubuntu


这样就删除成功了
